# 收势设计 · 实测记录

## 火把节 Clip 4 · 第一次实测

**旁白**：人们用松木和松脂制作火把，保证火焰燃烧得又稳又亮。

### 原版 Prompt（无收势）
- 描述「制作中」的静态画面
- 结尾：动作进行中被截断 → 突然中断感

### 新版 Prompt（收势设计）
```
@[Image]，场景保持图片一致。
[00-02s] 中景，阳光明媚的乡村庭院，男子盘腿坐在石板地上专注地用柴刀削松木棍，女子蹲坐在旁观察，微笑交流。
[02-04s] 男子将削好的木棍递给女子，女子用松脂绑缚加固，火把雏形完成，两人配合默契。
[04-06s] 男子举起完成的火把，展示成果，火把顶部草束整齐排列。
[06-08s] 男子用火绒点燃火把，火焰稳定燃起，火光照亮两人脸庞，镜头缓缓拉远，两人对视露出满足笑容。
旁白：人们用松木和松脂制作火把，保证火焰燃烧得又稳又亮。
整体风格保持温暖插画风格，色彩饱和，叙事完整有收尾感。
```

**收势设计**：
- 起（00-02s）：男子削木棍，女子观察
- 承（02-04s）：配合制作，火把雏形完成
- 转（04-06s）：举起完成的火把展示
- 合（06-08s）：点燃火把，火焰稳定燃烧，两人对视笑，镜头拉远 → **收势在「完成」**

**任务 ID**：`cgt-20260526182027-6hw2w`

---

## 三种收势模式

| 模式 | Clip 结尾状态 | 适用旁白类型 |
|------|-------------|-------------|
| **完成式收势** | 动作完成/达到目的/稳定状态 | 描述完整动作的旁白 |
| **进行式收势** | 运动中但小目标达成 | 描述持续状态的旁白 |
| **悬念式收势** | 动作未完成但情绪平稳 | 需要连接下一叙事单元 |

---

## 关键发现

1. **收势必须在 Prompt 里设计**，不能靠剪辑补救
2. **每张静态图 = 1个叙事单元 = 1个Clip**，不跳步单独测试
3. **seedance `--return-last-frame` 参数不接受值**，会报 `expected one argument`，需要单独研究 CLI 行为
